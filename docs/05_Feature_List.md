# ✅ 05. 기능 명세 및 현황 (Feature List & Status)

> 모든 기능은 고유 ID를 가지며, 우선순위와 현재 상태를 추적합니다.

### 상태(Status) 정의

- **`대기 (Backlog)`**: 개발 예정인 기능
- **`진행 중 (In Progress)`**: 현재 개발이 진행 중인 기능
- **`완료 (Done)`**: 개발 및 기본 테스트가 완료된 기능
- **`보류 (On Hold)`**: 우선순위 또는 정책에 의해 개발이 보류된 기능
